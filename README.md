## 지원 기능

- 핑 체크
- 특정 채널에 익명 메시지 남기기
- 특정 메시지에 스레드 남기기
- 메시지 실수로 보내지 않게 체크박스로 한 번 더 확인하기
- 메시지 별로 이름 남기기
- (그리고 약간의 재미를 위한) 익명 이름 생성기
